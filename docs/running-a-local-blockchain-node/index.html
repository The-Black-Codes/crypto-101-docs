Running a local Blockchain Node with Hardhat

What is a Blockchain Node?â€‹

A Blockchain Node is a computer that runs the blockchain. It is responsible for storing the blockchain data, and for running the consensus algorithm.

Running a Local Blockchain Node with Hardhatâ€‹

npx hardhat node

Deploying a Smart Contractâ€‹

At this point you have a local blockchain node running. Let's deploy a smart contract.

Your deploy scriptâ€‹

Create a file in the scripts directory called deployCounter.ts with the following contents:

import "@nomiclabs/hardhat-ethers";
import { ethers } from "hardhat";

async function deploy() {
  const Counter = await ethers.getContractFactory("Counter");
  const counter = await Counter.deploy();
  await counter.deployed();
}

deploy().catch(console.error);

Using hardhat to deploy your scriptâ€‹

npx hardhat run ./scripts/deployCounter.ts --network localhost
